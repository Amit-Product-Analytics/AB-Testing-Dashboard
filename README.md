# A/B Testing Analytics Dashboard

## Project Overview

This project compares a Control Variant and a Test Variant to determine which version performs better.

The objective of this analysis was to measure conversion performance, revenue impact, and user behavior in order to support data-driven product decisions.

---

## Business Problem

Businesses frequently run experiments to improve product performance and customer conversion.

This project answers the following questions:

* Which variant achieved the higher conversion rate?
* Which variant generated more revenue?
* What was the conversion lift?
* Which device type performed best?
* Which country achieved the highest conversion rate?
* Should the Test Variant be launched?

## Key Insights

* Test Variant achieved a higher conversion rate than the Control Variant.
* Test Variant generated higher revenue.
* Mobile users showed stronger response to the experiment.
* UK users recorded the highest conversion performance.
* Conversion Lift reached 49%.
